# Исправление размытых изображений на мобильных - Краткая сводка

## ✅ Что было исправлено

### 1. Добавлены CSS правила для четкости изображений
- `image-rendering: -webkit-optimize-contrast` - улучшает четкость на iOS
- `transform: translateZ(0)` - включает GPU-ускорение
- `backface-visibility: hidden` - предотвращает размытие при трансформациях

### 2. Улучшена загрузка фоновых изображений
- Добавлена предзагрузка изображений
- Добавлено логирование для отладки
- Оптимизирован переключатель мобильных/десктопных версий

### 3. Добавлен debounce для resize событий
- Предотвращает множественные перерисовки
- Улучшает производительность при изменении ориентации

### 4. Обновлены мета-теги
- Добавлены теги для улучшения отображения на iOS и Android
- Оптимизирован viewport для мобильных устройств

### 5. Мобильные изображения задеплоены

- `hero_mob.png` (446 KB)
- `ABOUT_mob.png` (196 KB)

## 🔍 Как проверить

1. **Откройте сайт на мобильном устройстве**: https://smt-avto.ru
2. **Или используйте Chrome DevTools**:
   - Нажмите F12
   - Нажмите Ctrl+Shift+M (Device Toolbar)
   - Выберите iPhone или другое мобильное устройство
   - Обновите страницу (Ctrl+R)

3. **Проверьте консоль** (F12 → Console):
   Вы должны увидеть:
   ```
   ✅ Hero background loaded: /images/works/hero_mob.png
   ✅ Background loaded: /images/works/ABOUT_mob.png
   ✅ Background loaded: /images/works/services_mob.png
   ✅ Background loaded: /images/works/sertif_mob.png
   ```

4. **Проверьте Network** (F12 → Network):
   - Убедитесь, что загружаются файлы с `_mob.png`
   - Проверьте размер загруженных файлов

## 🚀 Дальнейшие улучшения (опционально)

Если изображения все еще недостаточно четкие на Retina-дисплеях:

1. **Создать изображения @2x** (удвоенное разрешение):
   - hero_mob@2x.png: 1280x800px
   - ABOUT_mob@2x.png: 800x600px
   - services_mob@2x.png: 600x800px
   - sertif_mob@2x.png: 600x800px

2. **Конвертировать в WebP** для уменьшения размера:
   ```bash
   cwebp -q 85 hero_mob.png -o hero_mob.webp
   ```

3. **Использовать AVIF** для еще лучшего сжатия (на 50% меньше):
   ```bash
   avifenc --min 0 --max 63 hero_mob.png hero_mob.avif
   ```
