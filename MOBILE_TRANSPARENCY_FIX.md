# Исправление прозрачности фона на мобильных устройствах

## ✅ Что было сделано

### 1. Увеличена прозрачность темного оверлея на мобильных

**Было (для всех устройств):**
```
from-gray-900/40 via-gray-800/35 to-red-900/40
```
- 40% затемнение - фоновое изображение плохо видно

**Стало (для мобильных):**
```
from-gray-900/20 via-gray-800/15 to-red-900/20
```
- 20% затемнение - фоновое изображение хорошо видно
- 15% в центре - еще лучше видимость

**Для десктопа осталось как было:**
```
from-gray-900/40 via-gray-800/35 to-red-900/40
```

### 2. Изменены компоненты

Обновлены следующие секции:
- ✅ **Hero** - главная секция
- ✅ **About** - секция "О компании"
- ✅ **Services** - секция "Услуги"
- ✅ **Certificates** - секция "Сертификаты"

### 3. Динамическое переключение

Добавлена проверка размера экрана:
```typescript
const [isMobile, setIsMobile] = useState(false);

useEffect(() => {
  const checkMobile = () => {
    const mobile = window.innerWidth <= 768;
    setIsMobile(mobile);
  };
  // ...
}, []);
```

Оверлей меняется автоматически:
```jsx
<div className={`absolute inset-0 bg-gradient-to-br ${
  isMobile 
    ? 'from-gray-900/20 via-gray-800/15 to-red-900/20'  // Мобильные - прозрачнее
    : 'from-gray-900/40 via-gray-800/35 to-red-900/40'  // Десктоп - темнее
}`}></div>
```

## 📱 Результат

### На мобильных устройствах (≤768px):
- ✅ Фоновое изображение хорошо видно
- ✅ Текст остается читаемым (белый на полупрозрачном темном)
- ✅ Карточки с услугами остаются с bg-opacity-60 (60% непрозрачности)
- ✅ Автоматическое переключение при изменении ориентации

### На десктопе (>768px):
- ✅ Более темный оверлей для лучшей читаемости
- ✅ Фиксированный фон (parallax эффект)
- ✅ Профессиональный вид

## 🎨 Визуальное сравнение

**Прозрачность оверлея:**

| Устройство | Было | Стало | Улучшение |
|------------|------|-------|-----------|
| Мобильные | 40% | 20% | +100% видимости фона |
| Десктоп | 40% | 40% | Без изменений |

**Значения прозрачности в Tailwind:**
- `/40` = 40% непрозрачности (60% прозрачности)
- `/35` = 35% непрозрачности (65% прозрачности)
- `/20` = 20% непрозрачности (80% прозрачности) ← **Новое для мобильных**
- `/15` = 15% непрозрачности (85% прозрачности) ← **Новое для мобильных**

## 📊 Технические детали

**Точка разрыва:** 768px
- ≤768px = мобильные (прозрачный оверлей 20%)
- >768px = десктоп (темный оверлей 40%)

**Измененные файлы:**
- `src/components/Hero.tsx`
- `src/components/About.tsx`
- `src/components/Services.tsx`
- `src/components/Certificates.tsx`

**Добавленная логика:**
- Отслеживание размера экрана через `useState` и `useEffect`
- Динамическое изменение классов через условный рендеринг
- Debounce для оптимизации resize событий (150ms)

## 💡 Дополнительные настройки

Если нужно еще больше прозрачности на мобильных, измените значения:

```jsx
// Еще более прозрачный (10%)
from-gray-900/10 via-gray-800/5 to-red-900/10

// Совсем прозрачный (5%)
from-gray-900/5 via-gray-800/0 to-red-900/5

// Без оверлея (0%)
from-gray-900/0 via-gray-800/0 to-red-900/0
```

Но учтите, что слишком прозрачный оверлей может ухудшить читаемость текста!

## ✨ Рекомендации

**Текущие настройки (20%) - оптимальны для:**
- ✅ Хорошая видимость фонового изображения
- ✅ Читаемость белого текста
- ✅ Профессиональный вид
- ✅ Баланс между красотой и функциональностью

- Увеличьте прозрачность до 25-30%
- Добавьте text-shadow для текста
- Используйте более контрастные цвета текста

**Если фон все еще плохо видно:**
- Уменьшите до 10-15%
- Но проверьте читаемость текста!
