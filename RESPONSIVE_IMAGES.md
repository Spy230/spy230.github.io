# Адаптивные изображения для мобильных устройств

## Что было сделано

Все изображения на сайте теперь полностью адаптивны для мобильных устройств. Вот основные улучшения:

### 1. CSS-правила (src/index.css)

- **max-width: 100%** - изображения никогда не выходят за пределы контейнера
- **height: auto** - сохраняются пропорции изображений
- **object-fit: cover** - правильное масштабирование в контейнерах
- **Адаптивные размеры кнопок** - на мобилках кнопки меньше
- **Оптимизация отступов** - на мобилках уменьшены отступы для галереи

### 2. Компонент Hero (src/components/Hero.tsx)

- Изменён `backgroundSize` с `100% 100%` на `cover` для лучшей адаптации
- На мобилках фон переключается с `fixed` на `scroll` для экономии памяти
- Минимальная высота на мобилках: 400px

### 3. Компонент Gallery (src/components/Gallery.tsx)

- Добавлены атрибуты `loading="lazy"` и `decoding="async"` для оптимизации загрузки
- Класс `max-w-full` для гарантии адаптивности
- Адаптивная сетка Swiper:
  - 640px: 1.2 слайда
  - 768px: 2 слайда
  - 1024px: 3 слайда
  - 1280px: 4 слайда

### 4. Компонент Certificates (src/components/Certificates.tsx)

- Ленивая загрузка изображений (`loading="lazy"`)
- Асинхронное декодирование (`decoding="async"`)
- Адаптивные размеры кнопок навигации
- Оптимизированные отступы для мобильных

### 5. Компонент About (src/components/About.tsx)

- Использован тег `<picture>` для адаптивных изображений
- Разные размеры для разных экранов:
  - 640px и меньше: 400px ширина
  - 1024px и меньше: 600px ширина
  - Больше 1024px: полный размер
- Ленивая загрузка с fallback на альтернативные изображения

### 6. Компонент Header (src/components/Header.tsx)

- Логотип оптимизирован для мобильных
- Адаптивные размеры текста (sm:text-xl)
- Прозрачный фон при скролле для лучшей видимости

## Мобильная оптимизация

### Точки разрыва (Breakpoints)

- **640px** - мобильные телефоны (iPhone SE, Galaxy A10)
- **768px** - планшеты в портретной ориентации
- **1024px** - планшеты в ландшафтной ориентации
- **1280px** - десктопы

### Производительность

- **Ленивая загрузка** - изображения загружаются только при необходимости
- **Асинхронное декодирование** - не блокирует основной поток
- **Оптимизированные размеры** - меньше данных для мобильных
- **Фиксированный фон на десктопе** - красивый эффект параллакса
- **Прокручиваемый фон на мобилке** - экономия памяти

## Тестирование

Для проверки адаптивности:

1. Откройте DevTools (F12)
2. Нажмите Ctrl+Shift+M (или Cmd+Shift+M на Mac)
3. Выберите разные устройства:
   - iPhone SE (375px)
   - iPhone 12 (390px)
   - iPad (768px)
   - iPad Pro (1024px)

## Поддерживаемые браузеры

- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- iOS Safari 14+
- Android Chrome 90+

## Дополнительные рекомендации

1. **Оптимизация изображений** - используйте WebP или AVIF форматы
2. **Кэширование** - настройте кэширование на сервере
3. **CDN** - используйте CDN для быстрой доставки изображений
4. **Сжатие** - сжимайте изображения перед загрузкой на сервер

## Метатег Viewport

В `index.html` уже установлен правильный метатег:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

Это гарантирует правильное отображение на мобильных устройствах.
