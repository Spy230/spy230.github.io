# Добавлено изображение с вывеской на мобильную версию

## ✅ Что добавлено

### 1. Изображение в Hero секции (главная страница)

**Где:** Между заголовком и описанием, только на мобильных устройствах (≤768px)

**Изображение:** `/images/works/samotcvet.jpg` - фото автосервиса с вывеской "САМОЦВЕТ"

**Код:**
```jsx
{isMobile && (
  <div className="mb-8 max-w-md mx-auto">
    <img
      src="/images/works/samotcvet.jpg"
      alt="АвтоТехЦентр Самоцвет - наша вывеска"
      className="rounded-xl shadow-2xl w-full border-2 border-red-500"
      loading="eager"
    />
  </div>
)}
```

**Особенности:**
- ✅ Показывается только на мобильных (≤768px)
- ✅ Красная рамка (border-2 border-red-500)
- ✅ Тень (shadow-2xl)
- ✅ Скругленные углы (rounded-xl)
- ✅ Максимальная ширина 448px (max-w-md)
- ✅ Загружается сразу (loading="eager")

### 2. Изображение в About секции (О компании)

**Где:** В секции "О компании", адаптивное переключение

**Логика:**
- **Мобильные (≤768px):** показывается `samotcvet.jpg` (фото с вывеской)
- **Десктоп (>768px):** показывается `photo_2025-12-25_13-09-14.jpg` (мастер с автомобилем)

**Код:**
```jsx
<picture>
  <source 
    media="(max-width: 768px)" 
    srcSet="/images/works/samotcvet.jpg" 
  />
  <img
    src="/images/works/photo_2025-12-25_13-09-14.jpg"
    alt="АвтоТехЦентр Самоцвет - наш автосервис"
    loading="eager"
    className="rounded-xl shadow-lg w-full border border-gray-700"
  />
</picture>
```

**Особенности:**
- ✅ Использует HTML5 `<picture>` элемент
- ✅ Автоматическое переключение по размеру экрана
- ✅ Fallback на мастера для десктопа
- ✅ Оптимизировано для производительности

## 💡 Дополнительные настройки

### Если нужно изменить размер изображения в Hero:

```jsx
// Меньше
<div className="mb-8 max-w-sm mx-auto">  // 384px

// Больше
<div className="mb-8 max-w-lg mx-auto">  // 512px

// На всю ширину
<div className="mb-8 mx-auto">  // 100%
```

### Если нужно изменить рамку:

```jsx
// Без рамки
className="rounded-xl shadow-2xl w-full"

// Белая рамка
className="rounded-xl shadow-2xl w-full border-2 border-white"

// Толще рамка
className="rounded-xl shadow-2xl w-full border-4 border-red-500"
```

### Если нужно показывать на всех устройствах:

```jsx
// Убрать условие isMobile
<div className="mb-8 max-w-md mx-auto">
  <img src="/images/works/samotcvet.jpg" ... />
</div>
```
