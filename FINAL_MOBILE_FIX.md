# Финальные исправления для мобильных устройств

## ✅ Что исправлено

### 1. Фон стал НАМНОГО прозрачнее

**Было:**
- Мобильные: 20% затемнение
- Десктоп: 40% затемнение

**Стало:**
- **Мобильные: 10% затемнение** (90% прозрачности!) 🔥
- **Десктоп: 30% затемнение** (70% прозрачности)

**Конкретные значения:**
```jsx
// Мобильные (≤768px)
from-gray-900/10 via-gray-800/5 to-red-900/10
// 10% затемнение по краям, 5% в центре

// Десктоп (>768px)
from-gray-900/30 via-gray-800/25 to-red-900/30
// 30% затемнение по краям, 25% в центре
```

### 2. Исправлено "падение" фона при открытии прайса

**Проблема:**
При нажатии "Показать все услуги" фоновое изображение скроллилось вниз вместе с контентом.

**Решение:**
- Добавлен `background-attachment: fixed` для секции Services на десктопе
- На мобильных остается `scroll` для производительности
- Фон теперь остается на месте при раскрытии прайса

**CSS:**
```css
#services {
  background-attachment: fixed; /* Фон зафиксирован */
}

@media (max-width: 768px) {
  #services {
    background-attachment: scroll; /* На мобильных - scroll для производительности */
  }
}
```

## 💡 Если нужно еще прозрачнее

Если фон все еще недостаточно виден, можно уменьшить до:

```jsx
// Почти прозрачный (5%)
from-gray-900/5 via-gray-800/0 to-red-900/5

// Совсем без затемнения (0%)
from-gray-900/0 via-gray-800/0 to-red-900/0
```

**НО:** При 0% затемнения текст может быть плохо читаемым на светлых участках фона!

## ⚠️ Важно

**Текущие настройки (10%) - оптимальны:**
- ✅ Отличная видимость фонового изображения
- ✅ Текст остается читаемым
- ✅ Баланс между красотой и функциональностью
- ✅ Фон не "падает" при раскрытии контента

**Если текст плохо читается:**
- Можно увеличить до 15-20%
- Или добавить text-shadow для текста:
  ```css
  text-shadow: 0 2px 4px rgba(0,0,0,0.8);
  ```
